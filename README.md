# Smartbay

Smartbay est une application qui permet de gérer ses locataires en un seul endroit.  
Si quelqu'un possède un bien immobilier en location, le bailleur peut installer l'application sur son smartphone, entrer le nom de son locataire, le prix de la location, la période et choisir la durée.  

Les informations sont affichées sur une **carte** à l'écran d'accueil, avec possibilité de **modifier** et **supprimer**. Lorsqu'un locataire est ajouté, un **calendrier (.ics)** est généré et des **rappels** peuvent être reçus 1 jour avant selon le choix de l'utilisateur. L'application inclut également un **mode nuit** et des **notifications push**.  

## Pages principales

- `index.html` → Page d'animation du logo (avec `index.css`)  
- `app.html` → Page d'accueil du projet (avec `app.css`)  
- `index.js` → Script JavaScript principal
